# Battery Monitor - Linux 电池监控软件需求文档

## 1. 项目概述

**项目名称**：Battery Monitor（电池监控器）  
**目标平台**：Linux Mint 22.3 / Ubuntu 24.04 及兼容发行版（x86_64）  
**项目类型**：系统托盘常驻应用 + 桌面小组件 + 开机自启动  
**交付物**：`.deb` 安装包（封装所有依赖）

---

## 2. 运行环境分析

| 项目 | 信息 |
|------|------|
| 系统 | Linux Mint 22.3 (Zena)，基于 Ubuntu 24.04 Noble |
| 架构 | x86_64 |
| 电池路径 | `/sys/class/power_supply/BAT0` |
| 电池厂商 | Sunwoda |
| 电池型号 | L19D4PH3 |
| 电池类型 | 锂聚合物 (Li-poly) |
| 设计容量 | 61 Wh |
| 当前满充容量 | 47.79 Wh (78.34%) |
| 循环次数 | 264次 |

## 3. 核心功能需求

### 3.1 系统托盘图标（必需）

- 常驻系统任务栏，显示电池图标
- 图标动态显示当前电量（分档显示：0-20%, 20-40%, 40-60%, 60-80%, 80-100%）
- 充电状态时图标带有充电标识
- 右键菜单：
  - 显示主窗口
  - 显示电量详情
  - 开机自启动开关
  - 退出程序
- 左键单击：显示快速信息弹窗

### 3.2 主窗口界面（必需）

#### 3.2.1 电池基本信息面板
- **当前电量百分比**：大字体显示，如 "23%"
- **电池状态**：充电中 / 放电中 / 已充满 / 未充电
- **电池型号**：Sunwoda L19D4PH3
- **电池技术**：锂聚合物

#### 3.2.2 功率信息面板
- **当前充放电功率**：以瓦特(W)为单位显示
- **电压**：以伏特(V)为单位显示
- **电流**：以安培(A)为单位显示

#### 3.2.3 电量预估面板
- **剩余电量预估时长**：
  - 放电时：显示 "预计还可使用 X 小时 Y 分钟"
  - 充电时：显示 "预计还需 X 小时 Y 分钟充满"
- **预估基于**：当前功率和剩余电量计算

#### 3.2.4 电池健康面板
- **设计容量**：61 Wh
- **当前满充容量**：47.79 Wh
- **电池健康度**：78.34%（当前满充容量 / 设计容量）
- **循环次数**：264次
- **电池寿命状态**：
  - >90%：优秀
  - 80-90%：良好
  - 70-80%：一般
  - <70%：需关注

#### 3.2.5 历史记录面板
- **上次断开充电时间**：显示最近一次拔掉充电器的时间
- **今日使用统计**：
  - 总充电时间
  - 总放电时间
  - 电量变化曲线（实时折线图）
- **历史数据图表**：
  - 支持查看 1小时/6小时/24小时/7天 的电量变化
  - 使用 GTK DrawingArea 绘制折线图
  - 数据存储在本地 JSON 文件

### 3.3 桌面小组件（必需）

- **小型浮动窗口**：显示当前电量百分比和状态
- **可拖动位置**：用户可自由拖动到桌面任意位置
- **可最小化到托盘**
- **透明度设置**：可调节窗口透明度
- **样式**：
  - 圆角矩形半透明背景
  - 电量数字 + 状态图标
  - 充电时显示闪电动画

### 3.4 通知与告警（必需）

- **低电量告警**：
  - 电量低于 20% 时弹出系统通知
  - 可设置告警阈值（10%/15%/20%/25%）
- **电量充满通知**：
  - 充到 100% 时弹出通知
  - 可关闭此通知
- **电池健康告警**：
  - 电池健康度低于 70% 时提示更换

### 3.5 设置功能（必需）

#### 3.5.1 通用设置
- 开机自启动开关
- 启动时最小化到托盘
- 主题选择（浅色/深色/跟随系统）

#### 3.5.2 通知设置
- 低电量告警开关
- 低电量告警阈值设置
- 充满电通知开关
- 电池健康告警开关

#### 3.5.3 刷新设置
- 数据刷新间隔（1秒/2秒/5秒/10秒/30秒）
- 默认：2秒

### 3.6 开机自启动（必需）

- 安装后自动配置 autostart
- 用户可在设置中开关
- 使用 `~/.config/autostart/` 机制

## 4. 技术实现方案

### 4.1 技术栈选择

| 组件 | 选择 | 理由 |
|------|------|------|
| 主语言 | Vala | 语法简洁，编译为原生代码，性能优秀 |
| GUI 框架 | GTK-3 | 系统原生，Mint 深度集成 |
| 系统托盘 | Ayatana AppIndicator | 原生支持，无需额外绑定 |
| 电池数据 | /sys/class/power_supply/ | 内核接口，无需额外依赖 |
| 编译器 | valac | Vala 官方编译器 |
| 构建系统 | Meson | 现代化构建系统 |
| 打包 | dpkg-deb | 生成 .deb 包 |

**性能预期**：
- 内存占用：~10MB
- 启动时间：< 0.5秒
- CPU 占用：< 0.5%（空闲时）

### 4.3 数据采集实现

**数据源**：`/sys/class/power_supply/BAT0/*`

**主要数据字段**：
- `capacity`: 电量百分比
- `status`: 充电状态 (Charging/Discharging/Full/Not charging)
- `power_now`: 当前功率 (微瓦)
- `voltage_now`: 当前电压 (微伏)
- `current_now`: 当前电流 (微安)
- `energy_now`: 当前能量 (微瓦时)
- `energy_full`: 满充电量 (微瓦时)
- `energy_full_design`: 设计容量 (微瓦时)
- `cycle_count`: 循环次数

### 4.4 打包方案

使用 Meson 构建 + dpkg-deb 打包：

**构建依赖**：
```
Build-Depends: meson, valac, libgtk-3-dev, 
               libayatana-appindicator3-dev
```

**运行依赖**：
```
Depends: libgtk-3-0, libayatana-appindicator3-1
```

**安装路径**：
- 主程序：`/usr/bin/battery-monitor`
- 图标：`/usr/share/battery-monitor/icons/`
- 桌面文件：`/usr/share/applications/battery-monitor.desktop`
- 自启动文件：`/etc/xdg/autostart/battery-monitor.desktop`
